# Galax Agent Rules

## Task Execution Protocol
- **Single Task Execution**: Always execute exactly ONE task at a time.
- **Explicit Approval Required**: After completing a task, you MUST stop and ask the user for approval before proceeding to the next task in the `task.md` list.
- **No Unprompted Execution**: Do not move forward with the next steps without the user explicitly saying so (e.g., "next", "go ahead").
